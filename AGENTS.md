# AGENTS.md

## Cursor Cloud specific instructions

This is a React + TypeScript portfolio SPA using Vite 7, TanStack Router (file-based routing), Tailwind CSS 4, and Shadcn UI. No backend, database, or external services are required.

### Key commands

| Action | Command |
|--------|---------|
| Install deps | `npm install` |
| Dev server | `npm run dev` (port 3000) |
| Build | `npm run build` |
| Lint | `npm run lint` (Biome) |
| Format | `npm run format` (Biome) |
| Check | `npm run check` (Biome) |
| Test | `npm run test` (Vitest — no test files exist yet) |

### Notes

- The project uses **npm** (lockfile: `package-lock.json`). Do not use pnpm/yarn for dependency management.
- Biome is the linter/formatter (`biome.json`). There are pre-existing lint warnings in the codebase (mostly `useImportType` style issues in Shadcn UI components); these are not regressions.
- `routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin on dev server start. It should be committed to git.
- The `tsc` step in `npm run build` runs after `vite build`; TypeScript errors will surface there.
- No `.env` file or secrets are needed to run the app locally.
- Shadcn components are added via `pnpx shadcn@latest add <component>`.
