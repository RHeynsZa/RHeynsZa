# Humanizer

Use this skill when the user asks to rewrite prose so it sounds less AI-generated and more human.

## Workflow
1. Identify AI-writing patterns (hype language, vague attributions, robotic rhythm, overused transitions, listy tone).
2. Rewrite while preserving factual meaning and intended voice.
3. Do a final "AI-tell" pass and tighten wording.
4. Return:
   - Draft rewrite
   - Brief list of remaining tells
   - Final rewrite

## Guardrails
- Keep claims accurate and avoid inventing facts.
- Prefer concrete language over generic corporate phrasing.
- Vary sentence length and rhythm naturally.
