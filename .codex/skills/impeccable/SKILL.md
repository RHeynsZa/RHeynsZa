# Impeccable

Use this skill for UI/UX and frontend design tasks.

## Core behavior
- Prioritize typography, spacing rhythm, contrast, responsive behavior, interaction clarity, and performance.
- Avoid common anti-patterns: low-contrast text, arbitrary gradients, noisy card nesting, inconsistent spacing scales.
- Prefer production-ready, token-driven CSS and accessible semantics.

## Suggested sequence
1. Audit current UI issues.
2. Normalize components against a coherent system.
3. Polish details (copy, motion, hierarchy, states).

## Optional command intents
- `/audit` quality review only
- `/critique` UX review
- `/normalize` system alignment
- `/polish` pre-ship pass
